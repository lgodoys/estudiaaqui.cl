Este es un ejemplo de cómo se pueden aplicar estilos en cascada (CSS) incrustando el formato en el código del mismo archivo

Este es un texto con saltos de linea usando <br>
para que se visualice en una estructura de parrafo.
El estilo incrustado se insertó directamente con
la estructura <p style="color:red;">este parrafo</style>
y las etiquetas <i> </i> y <b></b> para aplicar el
formato Bold (negritas) e Italic (cursivas) de esta últma parte

Esto nos muestra que podemos aplicar formato al texto
tanto al incrustar un CSS interno (usando la etiqueta <style></style>)
como incrustando formato en linea del parrafo escrito
o solo en partes del parrafo escrito.
